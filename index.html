Como Implementar no Seu Site Original

  1. Adicione este código ao seu site original para receber as alterações de tema:
    2. // Verificar se estamos no site original (não no iframe) if (window === window.parent) { window.addEventListener('message', (event) => { // Verificar a origem da mensagem para segurança if (event.origin !== 'https://app.veloon.com.br/embed/0838f5e4-c3a7-4014-9dce-3f7f6cb5c287') return; const themeData = event.data; // Aplicar o tema if (themeData.action === 'changeTheme') { // Atualizar variáveis CSS document.documentElement.style.setProperty('--primary', themeData.primary); document.documentElement.style.setProperty('--secondary', themeData.secondary); document.documentElement.style.setProperty('--background', themeData.background); document.documentElement.style.setProperty('--text', themeData.text); // Salvar no localStorage para persistência localStorage.setItem('siteTheme', JSON.stringify(themeData)); } }); // Carregar tema salvo, se existir const savedTheme = localStorage.getItem('siteTheme'); if (savedTheme) { const themeData = JSON.parse(savedTheme); document.documentElement.style.setProperty('--primary', themeData.primary); document.documentElement.style.setProperty('--secondary', themeData.secondary); document.documentElement.style.setProperty('--background', themeData.background); document.documentElement.style.setProperty('--text', themeData.text); } }
  2. Substitua 'https://seu-dominio.com' pelo domínio do seu painel de controle
  3. Certifique-se de que seu site usa variáveis CSS para as cores principais
  4. O painel de controle enviará as novas cores para seu site através de postMessage
